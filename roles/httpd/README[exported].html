Role httpd

This role deploys an httpd server, from source of from system binaries (in development).

Variables

Public vars

Vars that must be considered by the user of the role.

  • httpd_server_root - Directory under httpd will be deployed. Default: {{ ansible_env.HOME }}/httpd
  • httpd_version - Default: 2.4.25
  • httpd_port - Default: 8000
  • httpd_src - Where to keep src files. Default: {{ httpd_server_root }}/src
  • httpd_source - If true, install from source else install from binaries.

Private vars

Vars only interesting for developers of the role.

  • httpd_become: not httpd_source
